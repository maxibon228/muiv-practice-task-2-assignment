# Задание 4 — data_draw.py

Самостоятельный архив для репозитория GitHub.

## Запуск

Windows: запустите install_windows.bat, затем run_windows.bat.

Linux/macOS:

    python3 -m pip install -r requirements.txt
    python3 data_draw.py

## Тесты

Windows: запустите test_windows.bat.

Linux/macOS:

    python3 -m pytest -q

## Индивидуализация

Перед сдачей откройте config.py и заполните STUDENT_ID, если он не указан. Для фамилии на букву С цветовая схема по умолчанию: GnBu.
