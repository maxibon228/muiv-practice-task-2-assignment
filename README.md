# Практика — задание №2: Разработка системы первичной визуализации данных


## Состав проекта

- `dataset.py` — модуль задания №1: при импорте загружает `dataset.csv` в глобальную переменную `df`, при прямом запуске формирует `report.txt`.
- `data_scatter.py` — модуль задания №2: приложение на `tkinter` для построения точечной диаграммы по двум числовым колонкам.
- `dataset.csv` — полный исходный датасет на 150 строк.
- `report.txt` — отчёт, пересобранный по полному `dataset.csv`.
- `requirements.txt` — зависимости Python.
- `test_data_scatter.py` — базовые автотесты логики задания №2.
- `test_gui_smoke.py` — smoke-тест интерфейса: проверяет кнопки выбора осей и сохранение графика.
- `TEST_CHECKLIST.md` — ручной чек-лист перед сдачей.
- `screenshots/` — изображения графиков для вставки в отчёт.
- `UI_TRANSLATION_REPORT.md` — краткий отчёт о русификации интерфейса.
- `ui_translation.patch` — diff с изменениями по русификации.

## Русские подписи в интерфейсе

Внутренние имена колонок в `dataset.csv` оставлены без изменений, чтобы не ломать загрузку данных. В окне приложения пользователь видит русские подписи:

| Колонка в CSV | Подпись в интерфейсе |
|---|---|
| `platelets` | `Тромбоциты` |
| `serum creatinine` | `Креатинин сыворотки` |
| `serum sodium` | `Натрий сыворотки` |
| `creatinine phosphokinase` | `Креатинфосфокиназа` |
| `ejection fraction` | `Фракция выброса` |

Кнопка сохранения называется `Сохранить`.

## Быстрый запуск

```bash
python -m pip install -r requirements.txt
python dataset.py
python data_scatter.py
```

После запуска `dataset.py` в консоли должно быть видно начало отчёта с размером датасета:

```text
(150, 9)
```

После запуска `data_scatter.py` откроется окно приложения. В нём должны быть русские кнопки числовых колонок слева и снизу:

- `Тромбоциты`
- `Креатинин сыворотки`
- `Натрий сыворотки`
- `Креатинфосфокиназа`
- `Фракция выброса`

Категориальные колонки `sex`, `smoking`, `death` во втором задании не выводятся.

## Проверка

```bash
pytest -q
```

На Linux без активного графического дисплея:

```bash
xvfb-run -a pytest -q
```

Если `tkinter` не установлен, на Linux его можно установить системным менеджером пакетов, например:

```bash
sudo apt install python3-tk
```

## Что проверить вручную

1. Окно `data_scatter.py` открывается.
2. По умолчанию выбраны оси `x = Тромбоциты`, `y = Креатинин сыворотки`.
3. Кнопки слева меняют ось X.
4. Кнопки снизу меняют ось Y.
5. При каждом нажатии обновляются подписи осей и точки на графике.
6. Можно выбрать одну и ту же колонку на обе оси.
7. Кнопка `Сохранить` создаёт PNG-файл вида `graphHH_MM_SS.png`.
