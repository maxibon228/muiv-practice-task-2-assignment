# Чек-лист проверки задания №2

## Данные

- [ ] Файл `dataset.csv` находится в одной папке с `dataset.py` и `data_scatter.py`.
- [ ] При запуске `python dataset.py` выводится размер `(150, 9)`.
- [ ] Файл `report.txt` создан и соответствует полному датасету.

## Интерфейс `data_scatter.py`

- [ ] Программа запускается командой `python data_scatter.py`.
- [ ] Окно приложения открывается без ошибок.
- [ ] Заголовок окна: `Задание № 2: Разработка системы первичной визуализации данных`.
- [ ] Слева отображаются русские кнопки числовых колонок.
- [ ] Снизу отображается такой же список кнопок в том же порядке.
- [ ] Видимые кнопки: `Тромбоциты`, `Креатинин сыворотки`, `Натрий сыворотки`, `Креатинфосфокиназа`, `Фракция выброса`.
- [ ] Категориальные колонки `sex`, `smoking`, `death` не отображаются в кнопках.
- [ ] По умолчанию выбраны `x = Тромбоциты`, `y = Креатинин сыворотки`.
- [ ] При нажатии кнопки слева меняется ось X и график обновляется.
- [ ] При нажатии кнопки снизу меняется ось Y и график обновляется.
- [ ] При выборе одной колонки на обе оси точки ложатся на диагональ `x = y`.
- [ ] Подписи осей отображаются на русском и соответствуют выбранным кнопкам.
- [ ] Маркер точек — треугольник вверх `^`, так как рекурсивная сумма ID `70193935` равна `1`.
- [ ] Кнопка `Сохранить` создаёт файл `graphHH_MM_SS.png`.
- [ ] После закрытия окна программа завершает работу.

## Автотесты

- [ ] `pytest -q` проходит без ошибок на машине с GUI.
- [ ] На Linux без дисплея проверка выполнена через `xvfb-run -a pytest -q`.
