# Русификация интерфейса задания №2

В приложении `data_scatter.py` переведены видимые пользователю подписи интерфейса.
Внутренние имена колонок `dataset.csv` оставлены исходными, чтобы не нарушать работу `pandas.DataFrame`.

## Карта отображения

| Колонка в CSV | Подпись в интерфейсе |
|---|---|
| `platelets` | `Тромбоциты` |
| `serum creatinine` | `Креатинин сыворотки` |
| `serum sodium` | `Натрий сыворотки` |
| `creatinine phosphokinase` | `Креатинфосфокиназа` |
| `ejection fraction` | `Фракция выброса` |

Кнопка сохранения оставлена как `Сохранить`, что соответствует примеру интерфейса в методичке.

## Проверка

- Списки кнопок слева и снизу остаются одинаковыми.
- Левая группа кнопок меняет ось X.
- Нижняя группа кнопок меняет ось Y.
- Подписи осей и заголовок графика обновляются на русском языке.
- Сохранение создаёт файл вида `graphHH_MM_SS.png`.
- Тесты обновлены под русские подписи интерфейса.
